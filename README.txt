Gästebucheinträge
------------------

Project structure:
* gaestebuch/ - root folder
  * database/
    * gaestebuch.sql - secure copy database
  * documentation/ - short documentation of project
  * ressources/
    * css/ - stylesheets
    * db/ - copy of database and table
    * img/ - images used for the site
  * src/ - source code
  * GHANGELOG.txt - Shows previous versions and latest release
  * LICENSE.txt - project license
  * README.txt - project structure
